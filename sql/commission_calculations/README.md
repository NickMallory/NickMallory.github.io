# Tiered Commission Calculations

This project demonstrates how tier-based commission rules
are applied to sales transactions for both agents and managers.

## Key Concepts Demonstrated
- Tiered business rules
- Financial calculations
- CTE-based transformations
- Separation of agent and manager commissions

## Typical Use Cases
- Insurance commission reporting
- Sales performance analysis
- Management incentive tracking

## Tools
- SQL Server (T-SQL)
